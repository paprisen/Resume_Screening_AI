# Resume_Screening_AI

## Project Overview

Resume Screening AI is a Python-based application that automatically analyzes resumes, extracts important information such as skills, education, and experience, and ranks candidates based on their relevance to a given job description.

## Features

1. Upload resumes in PDF format

2. Extract candidate information
  
3. candidate information

4. Identify key skills from resumes

5. Match resumes with job descriptions

6. Calculate similarity scores

7. Rank candidates based on relevance

8. Generate candidate summary reports


## Tools Used

* Python 

* Streamlit

* PyPDF2

* Pandas
  
* Scikit-learn

## How It Works

1. Upload one or more resumes.

2. Enter a job description.

3. The system extracts skills and relevant information.

4. Similarity scores are calculated.

5. Candidates are ranked based on relevance.

6. Results are displayed in ranked order.

7. Sample Output

* Candidate Name
  
* Extracted Skills

* Match Score

* Ranking Position

## Future Improvements

## Conclusion

This Resume Screening AI project demonstrates the practical application of Artificial Intelligence in the recruitment process. The system analyzes resumes, compares them with job requirements, and generates a relevance score to identify suitable candidates.

By automating resume evaluation, the project improves efficiency, reduces manual screening efforts, and helps recruiters make faster decisions. The implementation of PDF parsing, text processing, and similarity matching provides a strong foundation for future enhancements such as skill extraction, candidate ranking, and advanced NLP-based analysis.
